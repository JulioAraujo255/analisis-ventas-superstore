# analisis-ventas-superstore
Análisis de ventas de una cadena retail.  Exploración de rentabilidad por categoría, región y segmento de clientes para identificar áreas de mejora.

## Dataset
- 9.994 registros de órdenes (2011–2014)
- Variables: ventas, beneficio, descuento, categoría, 
  región, segmento de cliente

## Herramientas
- Excel (limpieza, tablas dinámicas, dashboard)

## Hallazgos principales
- Furniture genera pérdidas con descuentos altos
- Technology es la categoría más rentable
- La región South tiene mayor tasa de pérdidas
